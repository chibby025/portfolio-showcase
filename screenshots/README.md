# Screenshots Directory

## How to Add Screenshots

1. Take screenshots of your WeWatch platform
2. Name them clearly:
   - `cinema-hall.png` - 3D cinema environment
   - `payment-modal.png` - Token purchase modal
   - `admin-dashboard.png` - Analytics dashboard
   - `chat-reactions.png` - Live chat interface
   - `ticket-purchase.png` - Event ticketing
   - `wallet-balance.png` - User wallet

3. Optimize for web (keep under 500KB each):
   ```bash
   # Use online tools or:
   convert screenshot.png -resize 1200x -quality 85 screenshot-optimized.png
   ```

4. Add to this directory

## Tips for Great Screenshots

- Use light theme if possible (better for portfolios)
- Show real-looking data (not empty states)
- Blur any sensitive info (user emails, transaction IDs)
- Take at 1920x1080 or higher resolution
- Show UI in action (not just static pages)
- Include cursor/highlights to show interactivity
